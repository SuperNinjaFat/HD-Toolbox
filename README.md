# HD-Toolbox

Toolbox full of tools for Spelunky HD

## Development

Run Dev Frontend

```console
# First time install deps
pnpm install

# Run dev server
pnpm dev
```

Run Tauri Dev

```console
cargo tauri dev
```

## Release Build

Build Release Frontend

```console
pnpm build
```

Build exe and bundles

```console
cargo tauri build
```
